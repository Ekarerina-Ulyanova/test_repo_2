## Overview

A user-friendly desktop tool that lets individuals set a budget, record and manage expenses, and view monthly summaries, all stored locally for reliable tracking. It offers an intuitive interface and safeguards against overspending, making personal finance management simple and accessible.

## Core features

1. **Graphical User Interface**: A Tkinterâ€‘based GUI that displays the current budget, allows users to add budgets and expenses, view a list of expenses, and remove selected expenses.
2. **Budget Management**: Functions to add, update, and retrieve the current budget amount, ensuring that expenses cannot exceed the available budget.
3. **Expense Tracking**: Ability to record expenses with amount, category, and description, display them in a listbox, and remove individual entries.
4. **SQLite Persistence**: All budget and expense data are stored in a local SQLite database, providing durability across application restarts.
5. **Input Validation & Error Handling**: Robust checks for numeric input, budget sufficiency, and selection of expenses, with user feedback via message boxes.
